Start:
docker-compose -f docker-compose-neo4j-enterprise-fixed.yml up -d

Stop:
docker-compose -f docker-compose-neo4j-enterprise-fixed.yml down

Stop and remove all data:
docker-compose -f docker-compose-neo4j-enterprise-fixed.yml down -v

Check cluster status:
docker ps --format "table {{.Names}}\t{{.Status}}\t{{.Ports}}"

View logs:
docker logs neo4j-core1
docker logs neo4j-core2
docker logs neo4j-core3

Restart cluster after crash:
docker-compose -f docker-compose-neo4j-enterprise-fixed.yml restart

View cluster health:
docker exec neo4j-core1 cypher-shell -u neo4j -p Password "CALL dbms.cluster.overview()"
